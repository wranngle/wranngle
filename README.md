### Selected Work

*Eval harness for ElevenLabs voice agents — deterministic scenarios, real latency budgets, real Sarah.*  
I test voice agents with deterministic synthetic transcripts, strict latency budgets (e.g., TTFB p95 ≤ 800ms), prompt versioning via git tags, and custom scoring rubrics. This repo shows the methodology and harness wired to my production agent.
